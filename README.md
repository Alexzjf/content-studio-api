# Video → Twitter Post (Local)

Chrome-розширення: відео з озвучкою → транскрипт (Whisper локально) → пост для X (Ollama локально).

**$0 · без API-ключів · все на вашому Mac**

## Що потрібно

1. **Google Chrome** (109+)
2. **Ollama** — [ollama.com](https://ollama.com)
3. **Node.js** — тільки для збірки Whisper (один раз)

## Встановлення

### 1. Ollama

```bash
# Встановіть Ollama з ollama.com, потім:
ollama pull llama3.2
ollama serve
```

Інші безкоштовні моделі для постів (краща якість тексту):

```bash
ollama pull mistral        # добрий для української
ollama pull qwen2.5:7b     # сильний для коротких текстів
ollama pull gemma2:9b      # близький стиль до Google
```

Змініть модель у налаштуваннях розширення.

### 2. Збірка Whisper (один раз)

```bash
cd twitter-post-extension
npm install
npm run build
```

Це створить `offscreen.js` (~15MB) з локальним Whisper.

### 3. Завантажити розширення в Chrome

1. Відкрийте `chrome://extensions`
2. Увімкніть **Режим розробника**
3. **Завантажити розпаковане** → оберіть папку `twitter-post-extension`

## Як користуватись

1. Переконайтесь, що `ollama serve` працює
2. Натисніть іконку розширення
3. Завантажте відео з озвучкою
4. **Згенерувати пост**
5. **Вставити на X.com** (відкрийте x.com → «Що нового?»)

Перший запуск: Whisper завантажить модель ~40MB (кешується в браузері).

## Архітектура

```
Відео → аудіо (Web Audio API)
     → Whisper tiny (браузер, offscreen)
     → транскрипт
     → Ollama (localhost:11434)
     → пост → x.com
```

## Якщо не подобається якість постів

Змініть модель Ollama у налаштуваннях. Рекомендації:

| Модель | Стиль |
|--------|-------|
| `llama3.2` | Універсальна, швидка |
| `mistral` | Живий, розмовний текст |
| `qwen2.5:7b` | Добра для коротких постів |
| `gemma2:9b` | Ближче до стилю Google |

Також змініть **Стиль поста** у налаштуваннях.

## Пізніше для інших користувачів

Архітектура готова до розширення:

- Додати провайдер `gemini` / `openrouter` у налаштуваннях
- BYOK — поле для API-ключа користувача
- Публікація в Chrome Web Store

Локальна версія залишиться як режим «без ключів».

## NotebookLM

Публічного API для notebooklm.google.com **немає** (тільки Enterprise за $$$).
NotebookLM використовує Gemini під капотом — найближчий безкоштовний аналог: `gemma2` або Gemini API (можна додати пізніше).

## Troubleshooting

| Проблема | Рішення |
|----------|---------|
| Ollama не підключається | `ollama serve` у терміналі |
| Модель не знайдена | `ollama pull llama3.2` |
| Порожній транскрипт | Перевірте звук у відео |
| Whisper не працює | Запустіть `npm run build` |
| Не вставляє на X | Відкрийте поле «Що нового?» на x.com |
