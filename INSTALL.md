# Content Studio for X — Chrome Extension

## Завантаження (ВАЖЛИВО)

Розширення потрібно завантажувати **тільки з цієї папки**:

```
/Users/olexander/Desktop/twitter-post-extension
```

**НЕ** з `crm-platform/twitter-post-extension` — там стара версія 1.0.1 без app.html.

## Перевірка версії

1. Відкрийте `chrome://extensions`
2. Увімкніть **Developer mode**
3. **Load unpacked** → виберіть папку `Desktop/twitter-post-extension`
4. Клік по іконці → у шапці має бути **v1.19.2**
5. У шапці: **EN | UA | Panel | Width**

## Панель + ширина

1. Відкрийте вкладку (YouTube, X, будь-який сайт)
2. Іконка розширення → **Panel**
3. Панель з’явиться **праворуч у вкладці** (не окреме вікно)
4. Повзунок **Width** одразу змінює ширину панелі (15–50%)

## Мова

Натисніть **UA** у верхній смузі. «Sources» → «Джерела».
